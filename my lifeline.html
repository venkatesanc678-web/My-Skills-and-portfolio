HTML
Hypertext Markup Language (HTML) is the standard markup language[a] for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.
CSS
CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation and styling of a document written in a markup language, most commonly HTML. It is a cornerstone technology of the World Wide Web, working alongside HTML (for content and structure) and JavaScript (for interactivity).
JavaScript
JavaScript (JS)[a] is a programming language and core technology of the Web, alongside HTML and CSS. It was created by Brendan Eich in 1995.[6] Ninety-nine percent of websites use JavaScript on the client side for webpage behavior.[10]
